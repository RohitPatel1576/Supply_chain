# Supply Chain Dashboard - Power BI

## Project Overview
This project is an interactive Supply Chain Analytics Dashboard built using Power BI. It helps analyze inventory, sales, shipping, supplier performance, and operational KPIs for better business decision-making.

## Features
- Inventory Analysis
- Supplier Performance Tracking
- Shipping & Delivery Insights
- Revenue & Sales Analysis
- KPI Monitoring Dashboard
- Interactive Visualizations

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## Key Insights
- Identified top-performing suppliers
- Tracked delayed shipments
- Monitored inventory levels
- Analyzed revenue trends and order performance
